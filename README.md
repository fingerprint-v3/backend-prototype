# Pocketbase Backend

To run from a server, you can use the following command. Note that without the `--http` part, it will only be available on localhost. You can also specify a different port if you want.

- `wget https://github.com/pocketbase/pocketbase/releases/download/v0.39.0/pocketbase_0.39.0_linux_amd64.zip`
- `unzip pocketbase_0.39.0_linux_amd64.zip`
- `./pocketbase serve --http="0.0.0.0:8090"`
